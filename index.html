SEC-01 // ANÁLISIS

EL DESAFÍO STEM

¿Cómo transformar la teoría abstracta en experiencia tangible? La enseñanza tradicional se ha enfocado en calcular sobre el papel. Este proyecto propone un enfoque real: crear, aplicar y fallar. Al fabricar cohetes, el alumno aplica el método científico construyendo su propio vehículo de aprendizaje.

SEC-02 // PROPULSIÓN

MOTORES Y FÍSICA

Validación de dos sistemas de baja potencia seguros para la ESO, eliminando propelentes comerciales:

[ QUÍMICA ]

Bicarbonato + Vinagre. Reacción ácido-base (neutralización). Genera CO2 a presión para el impulso. (2º/3º ESO)

[ FÍSICA ]

Aire comprimido. Leyes de Newton en estado puro para estudio de aerodinámica y caída libre. (4º ESO)

SEC-03 // TELEMETRÍA

SISTEMAS EMBEBIDOS

La ingeniería no es solo propulsión. Mediante microcontroladores Arduino UNO, los estudiantes integran electrónica y programación, creando sensores de luz y sistemas de registro de datos de vuelo en tiempo real.

SEC-04 // SOFTWARE

MISSION CONTROL

Entorno LMS diseñado en Python/SQL con IDE integrado. Permite programar las placas directamente desde el navegador y evaluar telemetría en directo.
