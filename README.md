# Del espacio al aula: Diseño y validación de una herramienta educativa STEM basada en ingeniería aeroespacial
En este repositorio se pueden encontrar archivos relacionados con el proyecto de investigación llevado a cabo por Máximo García-Risco Morán: Del espacio al aula: Diseño y validación de una herramienta educativa STEM basada en ingeniería aeroespacial. 
Se organiza de la siguiente manera: 
- Carpeta Pagina web, donde se encuentra alojado el código completo de la pagina web diseñada para el proyecto. Está pensado para ser ejecutado en un entorno local, y probado en un sistema MacOS 26.0. 
- Carpeta Pagina-exposición, donde se puede ver el código de la pagina web diseñada para la divulgación del proyecto en la web.
- Imágenes y vídeos, que contiene diversas fotos y videos relacionados con el proyecto
- Planos, en la cual se pueden consultar los planos de los cohetes diseñados para el proyecto.
- otros, donde se puede encontrar diversos materiales de igual interés para el proyecto.
